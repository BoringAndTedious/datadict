# datadict
## 数据字典
1. springboot
2. jdk1.8
3. sqlite
## 功能
1. 查看表及表备注信息
2. 查看表字段信息
3. 编辑表及字段备注信息
4. 生成实体类
5. 导出excel
6. 筛选数据(需要点击首页导出按钮左边的表名（笑）)
